# LeetMetric

A web application that displays LeetCode user statistics and coding progress.

## Features
- Search LeetCode users by username
- Displays solved Easy, Medium, and Hard problems
- Circular progress indicators
- Shows total solved questions
- Displays submission statistics
- Responsive and clean UI


## Technologies Used
- HTML
- CSS
- JavaScript
- LeetCode API
- CORS Anywhere Proxy

## Live Demo

🌐https://leet-metric-leet-code-progress-trac.vercel.app/

## How to Use

1. Open the Live Demo link.
2. Enter the configured LeetCode username (`lavichauhan123`).
3. Click the **Search** button.
4. View the LeetCode statistics.

## Note
1) This project uses a CORS proxy ( https://cors-anywhere.herokuapp.com ) for API requests. If the data does not load, the proxy service may be unavailable.
2) This project is currently configured with a fixed/demo LeetCode username and may display data for only one profile. Dynamic username support is under development.
3) LeetCode Username: lavichauhan123
